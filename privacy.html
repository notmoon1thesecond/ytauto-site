Privacy Policy

YTAUTO · Last updated 2026-07-09

YTAUTO ("the app") is a personal, single-user tool that uploads short videos to one YouTube channel on behalf of its owner. This policy explains what Google user data the app accesses and how it is used.

What data is accessed

The app requests the https://www.googleapis.com/auth/youtube.force-ssl OAuth scope for a single Google/YouTube account: the account of the app's owner. Through this scope, the app:

  • Uploads video files to that YouTube channel (videos.insert)
  • Posts a top-level text comment on videos it uploads (commentThreads.insert)

The app does not read, collect, or transmit any other data from the account — no analytics, no subscriber lists, no other channels' data, no data belonging to any user other than the app's owner.

How the data is used

Access is used exclusively to publish video content the owner has already produced to their own channel, and to post an accompanying comment. Nothing is used for advertising, profiling, or any purpose beyond this single publishing action.

How the data is stored

The OAuth refresh token issued after authorization is stored locally on the owner's own machine, in a file that is never committed to version control and never transmitted to any third party or server other than Google's own API endpoints. No Google user data is sold, shared, or disclosed to any other party.

Data retention and revocation

The stored token is retained only until the owner revokes access. Access can be revoked at any time from the Google Account's Security → Third-party access page, which immediately invalidates the token.
